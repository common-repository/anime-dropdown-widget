== Description ==

This widget shows all tags that are anime titles as a dropdown menu. It creates a taxonomy called Anime, you only need to tag your posts not in the 'Post Tags' box, but the 'Anime' box.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Tag your posts through the 'Anime' box on edit page
1. Place the widget into your sidebar through the 'Widgets' menu in WordPress
